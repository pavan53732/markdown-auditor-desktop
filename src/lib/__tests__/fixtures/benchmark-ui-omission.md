# AstraBuild UI Surface Contract Spec

## Functional Requirement
The system MUST provide a dedicated "Rollback" button in the primary execution dashboard.
This button MUST be active only when an execution is in progress or has failed.

## Component Definition
- Header
- Sidebar
- ExecutionLog
- ControlPanel
  - StartButton
  - StopButton
